**렌더 구조** (중요):
```
1. 헤더 (유저명, 시간, 삭제 버튼)
2. 본문 (댓글 내용)
3. 액션 (답글 버튼)
4. 답글 폼 (조건부)
5. 재귀: 대댓글 렌더 ← 핵심!

   │  └─ CommentItem (reply 1-1)  [재귀 계속...]
   └─ CommentItem (reply 2)
```

**재귀 구현**:

**스타일**:

## 💾 상태 관리 상세 설계

### 상태 흐름도
```
초기:
comments = []
loading = true
showReplyForm = {}

↓ API 로드
comments = [
  { id: 1, content: '댓글1', replies: [
    { id: 2, content: '답글1', replies: [] },
    { id: 3, content: '답글2', replies: [] }
  ]},
  { id: 4, content: '댓글2', replies: [] }
]
loading = false

↓ 사용자가 댓글 1의 "답글" 클릭
showReplyForm = { 1: true }

↓ 답글 입력 후 제출
새 Comment 객체: { id: 5, parentCommentId: 1, ... }
comments 업데이트: 댓글 1의 replies에 추가
showReplyForm = { 1: false }
```

### 불변성 패턴
```typescript
// 깊은 위치의 대댓글 추가 (재귀)
const addReplyRecursive = (
  arr: Comment[], 
  parentId: number, 
  newReply: Comment
): Comment[] => {
  return arr.map(c => ({
    ...c,
    replies: c.id === parentId
      ? [...c.replies, newReply]
      : addReplyRecursive(c.replies, parentId, newReply)
  }));
};

// 사용
const newComments = addReplyRecursive(comments, parentCommentId, newReply);
setComments(newComments);
```

### Step 4: API 서비스 (src/services/commentService.ts)
```typescript
export const commentService = {
  getComments: async (postId: number) => {
    const response = await api.get(`/posts/${postId}/comments`);
    return response.data;
  },

  createComment: async (postId: number, content: string) => {
    const response = await api.post(`/posts/${postId}/comments`, { content });
    return response.data;
  },

  createReply: async (parentCommentId: number, content: string) => {
    const response = await api.post(
      `/comments/${parentCommentId}/replies`, 
      { content }
    );
    return response.data;
  },

  deleteComment: async (commentId: number) => {
    await api.delete(`/comments/${commentId}`);
  },
};
```

### Step 5: 실제 API 연동 (백엔드 준비 후)
```typescript
// CommentSection에서
const fetchComments = async (postId: number) => {
  try {
    setLoading(true);
    const data = await commentService.getComments(postId);
    setComments(data.comments);
  } catch (err) {
    setError('댓글을 불러올 수 없습니다');
  } finally {
    setLoading(false);
  }
};
```
