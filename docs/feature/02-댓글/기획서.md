# 댓글/대댓글 기능 기획서

## 📋 개요
게시글 상세 페이지에 댓글 시스템을 추가합니다. 사용자가 게시글에 댓글을 달고, 댓글에 대댓글을 달 수 있는 중첩 구조의 댓글 기능입니다.

**상태**: Phase 1 계획 완료, 코드 구현 대기

---

## 🎯 기능 범위 (Phase 1)

### ✅ 필수 기능
- 댓글 작성 (게시글 상세 페이지)
- 댓글 목록 조회 (중첩 구조)
- 댓글 삭제 (본인 댓글만)
- 대댓글 작성 (댓글 하위)
- 대댓글 목록 (인덴트 표시)
- 대댓글 삭제 (본인만)

### ❌ Phase 2 기능
- 댓글 수정
- 댓글 좋아요
- 댓글 신고
- 페이지네이션

## 📊 데이터 모델

### Comment 인터페이스
```typescript
interface Comment {
  id: number;
  postId: number;
  content: string;
  author: {
    id: number;
    username: string;
    profileImage?: string;
  };
  createdAt: string;
  updatedAt: string;
  replies: Comment[];           // 재귀 구조 - 대댓글 배열
  parentCommentId?: number;     // null이면 최상위 댓글
}
```

### CommentsResponse 인터페이스
```typescript
interface CommentsResponse {
  comments: Comment[];  // 최상위 댓글만 (replies는 중첩)
}
```

**특징**:
- 재귀 구조로 무한 깊이의 중첩 가능
- 최상위 댓글만 API에서 반환 (replies에 자식들이 포함)
- parentCommentId로 계층 추적 가능
