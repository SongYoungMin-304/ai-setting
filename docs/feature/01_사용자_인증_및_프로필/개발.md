## 1. 개발 환경 설정

### 1.1 백엔드 환경
```bash
# JDK 21 설치 확인
java -version

# 프로젝트 셋업 (이미 backend-api 존재)
cd /Users/song-youngmin/Desktop/AI/claude/backend-api

# 의존성 설치
./gradlew build

# 서버 실행
./gradlew bootRun
# 또는 IDE에서 Run
```

### 1.2 프론트엔드 환경
```bash
# Node.js 18+ 설치 확인
node -v && npm -v

# 프로젝트 생성
npx create-react-app frontend --template typescript
cd frontend

# 필수 라이브러리 설치
npm install axios react-router-dom react-hook-form zod

# 개발 서버 실행
npm start
# http://localhost:3000
```

    fun signup(@RequestBody request: SignupRequest): ResponseEntity<AuthResponse>
    
    @PostMapping("/login")
    fun login(@RequestBody request: LoginRequest): ResponseEntity<AuthResponse>

  }

## 6. 주의사항

- [ ] 비밀번호는 절대 평문으로 로그에 출력하지 않기
- [ ] JWT 시크릿 키는 environment variable에서 로드
- [ ] CORS 설정을 개발 환경에만 적용 (운영에선 수정 필요)
- [ ] 모든 인증 필요 API는 @Secured 또는 @PreAuthorize 사용
- [ ] 데이터베이스 마이그레이션은 Flyway 또는 Liquibase 사용 고려
- [ ] API 응답에서 민감한 정보(비밀번호 등) 제외
