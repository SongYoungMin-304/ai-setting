## 2. Backend 구현 상세

### 2.1 CommentLike Entity (Kotlin)

```kotlin
@Entity
@Table(name = "comment_likes", uniqueConstraints = [
    UniqueConstraint(columnNames = ["comment_id", "user_id"], name = "uk_comment_user")
], indexes = [
    Index(name = "idx_comment_id", columnList = "comment_id"),
    Index(name = "idx_user_id", columnList = "user_id")
])
class CommentLike(
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    val id: Long? = null,

    @ManyToOne(fetch = FetchType.LAZY)
    @JoinColumn(name = "comment_id", nullable = false)
    val comment: Comment? = null,

    @ManyToOne(fetch = FetchType.LAZY)
    @JoinColumn(name = "user_id", nullable = false)
    val user: User? = null,

    @Column(nullable = false, updatable = false)
    val createdAt: LocalDateTime = LocalDateTime.now(),

    var updatedAt: LocalDateTime = LocalDateTime.now()
)
```

### 2.2 CommentLikeService (Kotlin)

**주요 메서드:**
- `addLike(commentId, userId)`: 좋아요 추가
- `removeLike(commentId, userId)`: 좋아요 취소
- `isLiked(commentId, userId)`: 좋아요 여부 확인
- `getLikeCount(commentId)`: 좋아요 수 조회

**예외 처리:**
- `자신의 댓글에 좋아요할 수 없습니다`
- `이미 좋아요한 댓글입니다`
- `댓글을 찾을 수 없습니다`
- `좋아요 기록을 찾을 수 없습니다`

### 2.3 CommentLikeController (Kotlin)

**엔드포인트:**
- `POST /comments/{commentId}/likes`: 좋아요 추가 (201 Created)
- `DELETE /comments/{commentId}/likes`: 좋아요 취소 (200 OK)

**응답 포맷:**
```kotlin
data class CommentLikeResponse(
    val commentId: Long,
    val likeCount: Long,
    val isLiked: Boolean
)
```

## 5. 개발 과정에서 발생한 이슈

### Issue 1: axiosInstance import 오류
**원인:** api.ts에서 `api`로 export했는데 `axiosInstance`로 import 시도

**해결:** import 명 수정
```typescript
// 변경 전: import { axiosInstance } from '../services/api';
// 변경 후: import api from '../services/api';
```

### Issue 2: Comment 타입 미정의
**원인:** 기존 Comment 타입에 likeCount, isLiked 필드 없음

**해결:** types/index.ts에 필드 추가
```typescript
likeCount?: number;
isLiked?: boolean;
```

### Issue 3: CommentItem 레이아웃
**원인:** 좋아요 버튼과 답글 버튼의 배치 필요

**해결:** flex 레이아웃으로 정렬
```tsx
<div className="flex items-center gap-3">
  <CommentLikeButton ... />
  <button>답글</button>
</div>
```
