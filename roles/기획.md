# 기획 (Planning)

## 프로젝트 개요
- **목표**: Kotlin 백엔드 + React 프론트엔드로 구성된 풀스택 애플리케이션 개발
- **기간**: TBD
- **팀**: 백엔드 개발자, 프론트엔드 개발자, QA 엔지니어

## 기술 스택
### Backend
- **언어**: Kotlin
- **프레임워크**: Spring Boot
- **DB**: PostgreSQL / MySQL
- **API**: RESTful API

### Frontend
- **언어**: JavaScript/TypeScript
- **프레임워크**: React
- **상태 관리**: Redux / Zustand / Context API
- **UI 라이브러리**: Material-UI / Tailwind CSS
